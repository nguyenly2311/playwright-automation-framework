# Playwright Automation Framework

A Playwright + TypeScript automation framework following the Page Object Model (POM) design pattern.

This project was built to practice scalable test automation architecture with reusable page objects, fixtures, storage state authentication and maintainable test structure.

---

## Tech Stack

- Playwright
- TypeScript
- Node.js

---

## Project Structure

```text
.
├── components/
├── constants/
├── fixtures/
├── pages/
├── test-data/
├── tests/
│   ├── inventory/
│   └── login/
├── utils/
├── playwright.config.ts
└── package.json
```

- Page Object Model (POM)

## Installation

```bash
npm install
```

Install Playwright browsers

```bash
npx playwright install
```

---

## Run Tests

Run all tests

```bash
npx playwright test
```

Run in headed mode

```bash
npx playwright test --headed
```

Run a specific test

```bash
npx playwright test tests/inventory/filter.verification.spec.ts
```

---

## Report

Open Playwright HTML Report

```bash
npx playwright show-report
```

---

## Authentication

Authenticated test cases reuse Playwright Storage State instead of logging in before every test.

Authentication is generated by:

```text
tests/login/auth.setup.ts
```

---

## Screenshots

A screenshot is automatically captured after every test execution.

Failed tests also include:

- Screenshot
- Video
- Trace
